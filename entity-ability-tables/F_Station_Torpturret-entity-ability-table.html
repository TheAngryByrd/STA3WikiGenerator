==Abilities==
Name Description
[[Heavy Torpedo Salvo]] Worried that the M-23 would not be able to adequately defend against Dominion or Borg attacks, Starfleet Command has ordered all M-23 Torpedo Turrets to be upgraded with a multiple-launch system that allows the capability of a full spread of torpedoes to be fired in one salvo.
[[Proximity Torpedo Mode]] EFFECT: Fires torpedoes that have a wider area of effect. Starfleet's Corps of Engineers have outfitted all M-23 Turrets with the capability to launch proximity detonating torpedoes designed to strike multiple hostile targets within an enveloping blast radius.
