==Description== Designation: Vessel 3114 Analyze Threat Profile: Cruiser Classes - Vessel type will assist in assimilating hostile species. ==Abilities==
Name Description
[[Auto Assimilator]] EFFECT: Damages an ensnared enemy while repairing the Adaptor. Activate autobeams. Dispatch tactical drones. Targets marked for immediate assimilation will be assimilated.
[[Engine Overload Pulse]] EFFECT: Disables enemy engines for a time. Adapt deflector for overload pulse. Disable target engines. Assimilate.
[[Shield Overload Pulse]] EFFECT: Renders enemy shields less effective for a time. Adapt deflector for overload pulse. Disrupt target shields. Assimilate.
[[System Integrity]] As Systems are damaged, they will become less effective.
[[Uncapturable]] Due to impenetrable security, command of this vessel cannot be taken by storm.
